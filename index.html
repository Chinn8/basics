Paragraph Tags

This is a standard paragraph. The paragraph tag (p) is one of the most basic HTML elements. It represents a paragraph of text.

This is a highlighted paragraph with a special class applied. CSS can be used to style specific elements using classes and IDs.

HTML allows you to create strong (bold) text and emphasized (italic) text using the appropriate tags.

Image Tag

The image above is a placeholder. The img tag requires src (source) and alt (alternative text) attributes.

Video Tag

The video tag allows you to embed videos in your HTML document. The controls attribute adds play/pause buttons.

Audio Tag

The audio tag embeds sound content in your HTML document. Like the video tag, it often uses the controls attribute.
